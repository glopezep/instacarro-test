# incidents-api

## Usage

```sh
npm run api
```

## API

### POST /incidents
Create an incident

### GET /incidents
Retrieve an list of incidents

### POST /incidents/:incidentId/archive
Archive an incident

### POST /localities
Create a locality

### GET /localities
Retrieve a locality by id

### GET /localities/:localityId
Retrieve an list of localities
